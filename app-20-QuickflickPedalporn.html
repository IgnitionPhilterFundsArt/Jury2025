20 - Quickflick Pedalporn

Description for the Jury: Explain the Project in Full. [This is the description that should explain why the jury should give you the funds so try to answer any questions they might have. Please include some language on how the Flipizens will interact with the piece.]

It is a hand cranked pedal powered kinetoscope. The participant has to adjust the rotation and the strobe light speeds at the same time to be able to watch the movie on the screen. The art piece consists of a strobe light and a disassembled bicycle. The hand cranked chain rotates a wheel which holds the frames of the 3-4 second film on the rim. An adjustable strobe light projects the images of a unsavory porn on to a screen. If you get the rhythm perfect, it is non-stop fun.
