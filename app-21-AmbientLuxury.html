21 - Ambient Luxury

Description for the Jury: Explain the Project in Full. [This is the description that should explain why the jury should give you the funds so try to answer any questions they might have. Please include some language on how the Flipizens will interact with the piece.]

I'm revisiting an art piece concept I did back in 2018 for FS. I get an antique window and then do a stained glass mosaic over the top of it. I then cut out a backing board for the window, and make an LED matrix on it, add a diffusion layer, attach a chain, and hang from a hammock chair stand. The window looks like it is floating and catches all the hippies. I plan to have a small speaker playing the finest of ambient music. Like challenging ambient music, not just downtempo. I plan on having a few push buttons on hand that can trigger some different patterns on the piece, some of which will be sound reactive. I would hang this new piece beside the one I did in 2018--which is shown in the photo I included. Ultimately, I'd like to have six or eight of these windows so I can make a glowing stained glass room at Flipside.
