8 - Flipside 500

Description for the Jury: Explain the Project in Full. [This is the description that should explain why the jury should give you the funds so try to answer any questions they might have. Please include some language on how the Flipizens will interact with the piece.]

I want to bring my race track mixed reality game to Flipside. I have set up a couple RC cars with steering wheels and pedals that have a live view projected onto a screen. I plan to have it working with 2 players simultaneously, and have prizes for fastest lap times as well as some other more fun prizes. The track will be lined with LEDs that are synced to the lap timing system. I will have a leaderboard on display whenever possible. The event is all contained and run inside a walled 10X10 EZ-up, with the projection going onto a white wall so people outside can watch while others race. The event will only be run when it is dark, because the projections do not show up very well in the day.
